# SoulGolem

Paper/Folia плагин: медные големы **Soul Farmer** и **Soul Miner** — ферма и добыча в личной зоне с сундуком, забором и защитой территории.

**Требования:** Paper 1.21.9+, Java 21  
**Folia:** поддерживается (`folia-supported: true`)

## Быстрый старт

1. Скрафти статую (как сундук): **8 медных слитков** по кругу + в центре **семена пшеницы** (фермер) или **уголь** (майнер) — или `/soulgolem give …`
2. ПКМ статуей по траве (блок из конфига)
3. Голем обустраивает зону: граница, сундук, забор, калитка (+ тропинка под ней)
4. Клади ресурсы в **Soul Chest** — дальше работает сам

Крафт статуй: `crafting.enabled` (и `crafting.farmer` / `crafting.miner`) в `config.yml`.

**Управление**
- Shift + ПКМ по голему — пауза / продолжить
- `/soulgolem list` / `gui` — список, стоп, резюм, ТП, удаление

## Soul Farmer

Пашет, сажает, собирает, носит лут в сундук.

| В сундук | Зачем |
|----------|--------|
| Семена пшеницы / моркови / картошки | Посадка |
| Верстак | Ставит рядом с сундуком; без него — только сбор пшеницы |
| Костная мука | Удобрение незрелых культур |
| Компостер | Ставит рядом с верстаком (или сундуком); лишние семена/урожай → костная мука |
| Факела | Углы фермы |
| Ступени | Лавочка на границе |
| Забор / калитка | Любой тип из сундука (дуб/ель/незер…); в конфиге — предпочтение |
| Пшеница (3+) | Печёт хлеб, если верстак уже стоит |
| Корм (энергия) | Когда голоден |

### Приоритет задач (сверху важнее)

1. Калитка / дождь (навес)
2. Бой (если есть меч/топор и мобы в зоне)
3. Сбор урожая
4. Сорняки на поле
5. Вспашка
6. **Посадка** (пустые грядки + семена → идёт в сундук за семенами, потом сажает)
7. Хлеб (если есть пшеница)
8. Удобрение костной мукой
9. Чистка границы / лавочка / факела

«Ждёт семена» — только если поле пустое и семян нет. Если уже что-то растёт — «смотрит за грядками» и занимается хлебом/мукой/прочим.

## Soul Miner

Копает руду в зоне, носит в сундук, факела / лавочка / забор как у фермера.

| В сундук | Зачем |
|----------|--------|
| Железная / алмазная / незеритовая кирка | Апгрейд (больше блоков за рейс) |
| Факела, ступени, забор / калитка | Обустройство, тропинка под калиткой |
| Корм | Энергия |

---

## Бой (оба типа)

Меч или топор:
- ПКМ по голему с оружием в руке
- кинуть на землю в зоне
- положить в сундук

Голем берёт оружие и бьёт мобов **только внутри шерстяной границы** (за забор не уходит). После зачистки оружие возвращает в сундук.

Конфиг: секция `combat` в `config.yml`.

---

## Защита зоны

Вся территория голема защищена (и после рестарта).

- Чужие и владелец **не ломают** зону (кроме исключений владельца)
- Владелец может: открывать сундук, ломать урожай, вытаптывать грядки, открывать калитку
- Верстак голема закрыт для всех по умолчанию (`farmer.denyCraftOpen`) — в т.ч. владельца; выключить в конфиге при необходимости
- Сундук и верстак станции не ломаются (админ: Creative + Shift)
- Удаление голема — через GUI

## Команды

| Команда | Право | Описание |
|---------|--------|----------|
| `/soulgolem give <игрок> <miner\|farmer> [кол-во]` | `soulgolem.give` | Выдать статую |
| `/soulgolem reload` | `soulgolem.reload` | Перезагрузить конфиг |
| `/soulgolem list` / `gui` | `soulgolem.use` | GUI управления |

Алиасы: `/sg`, `/sgol`

`soulgolem.admin` даёт доступ ко всем подкомандам. Tab предлагает только те, на которые есть право.

| Право | default | Описание |
|-------|---------|----------|
| `soulgolem.admin` | op | Все команды + обход лимита |
| `soulgolem.use` | true | GUI, размещение и управление своими големами |
| `soulgolem.give` | op | Выдача статуй |
| `soulgolem.reload` | op | Reload конфига |
| `soulgolem.bypass.limit` | op | Обход личного лимита големов |

## Конфиг

| Путь | Содержимое |
|------|------------|
| `config.yml` | Лимиты, энергия, фермер/майнер, бой, БД, permissions |
| `gui/*.yml` | Слоты и материалы GUI |
| `lang/ru.yml`, `lang/en.yml` | Тексты (`language` в config) |

Дефолты заданы в Java (Elytrium serializer) — fresh install работает без ручной дописки ключей.
